# ⚡ ShopAI: The Future of E-Commerce

**ShopAI** is a premium, AI-first e-commerce experience designed to redefine how users discover and shop for products. Built with a minimalist "Studio" aesthetic, it leverages AI to provide personalized recommendations, sentiment analysis, and a seamless shopping journey.

![ShopAI Preview](https://raw.githubusercontent.com/ROHITH-NATHANI/ShopAI/main/public/images/hero_bg_1.png)

## ✨ Features

- **🤖 AI-First Personalization**: 
  - **AI Style Quiz**: Interactive onboarding to find your "Style Profile."
  - **AI Sentiment Insights**: Procedural analysis of product reviews to help you decide.
  - **Smart Bundle Logic**: AI suggestions for complementary products in the chat.
- **💎 Premium UI/UX**:
  - **Light/Dark Mode Persistence**: Automatically saves your theme preference.
  - **Fluid Animations**: Powered by `Framer Motion` for a high-end feel.
  - **Shimmer Loading**: Smooth skeleton states for image placeholders.
  - **Cinematic Hero**: A sliding multi-background hero section.
- **🛠️ Robust Architecture**:
  - **Persistence**: Cart and Wishlist are saved in `localStorage`.
  - **Semantic Search**: AI-powered command bar for natural language queries.
  - **Tech Stack Dashboard**: Interactive visualization of the project's architecture.

## 🛠️ Tech Stack

- **Frontend**: React.js, Framer Motion, Lucide React
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (Custom UI System)
- **AI**: Claude AI (Anthropic API Integration)
- **State**: React Hooks & LocalStorage

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/ROHITH-NATHANI/ShopAI.git
cd ShopAI
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```

### 4. Build for production
```bash
npm run build
```

## 📂 Project Structure

- `src/App.jsx`: Main application logic and styles.
- `public/images/`: High-resolution product and hero assets.
- `index.html`: Base template.
